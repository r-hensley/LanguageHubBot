This bot will record and log at a maximum (but potentially not all of):
- Number of messages per day per channel in your server
- Time of voice activity per day per voice channel in your server
- Number of members in your server per day
- An invite link to your server
- Other general meta information about your server (name, server ID, etc)

This bot will NOT ever record or read:
- Message content (it will not even receive this from the API; all messages it will receive will not have message content included)
- User names, IDs, or any other identifiable information about your members of any kind.

If you wish to remove the data about your server that this bot records, please remove it from your server and it'll automatically remove all data it has about you. Note that if your server is part of the language hub list, this may remove your server from the list.
